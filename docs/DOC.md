# MacPeripheralHub

MacPeripheralHub - macOS-приложение версии 1.0.0 для контроля подключенной периферии, быстрого переключения активных устройств и удержания выбранных системных устройств по умолчанию.

## Суть приложения

- Показывает все подключенные к Mac периферийные устройства в одном месте.
- Работает как обычное macOS-приложение с полноценным окном.
- После закрытия окна красной кнопкой приложение скрывается из Dock и продолжает работать в menu bar.
- В menu bar доступна компактная версия основных функций приложения.
- Через menu bar можно полностью завершить приложение кнопкой Quit.
- При первом запуске предлагает включить автозапуск при входе в macOS.
- Основная версия приложения: `1.0.0`.
- Название приложения и репозитория: `MacPeripheralHub`.

## Какие устройства показывает

- Мониторы и встроенные дисплеи.
- Наушники.
- Колонки.
- Микрофоны.
- Аудиоинтерфейсы и аудиокарты.
- Камеры и веб-камеры.
- Клавиатуры.
- Мыши.
- Трекпады.
- Bluetooth-устройства.
- USB-устройства.
- USB-хабы и док-станции.
- Неопознанные или частично определенные устройства.

## Какие характеристики показывает

- Название устройства.
- Категория устройства.
- Производитель, если доступен.
- Модель, если доступна.
- Серийный номер, если доступен.
- Тип подключения: USB, Bluetooth, встроенное, HDMI, DisplayPort, Thunderbolt, virtual, unknown.
- Для мониторов: разрешение, частота обновления, масштабирование, основной ли дисплей.
- Для аудиовыходов: количество каналов, частота дискретизации, текущий статус default output.
- Для микрофонов: количество каналов, частота дискретизации, текущий статус default input.
- Для системного аудиовыхода: текущий статус default system output.
- Для камер: имя, производитель, transport type, unique id, если доступно.
- Для HID-устройств: usage page, usage, vendor id, product id, transport.
- Для Bluetooth-устройств: адрес, имя, paired/connected статус, класс устройства, если доступно.

## Основные вкладки

- `Manual Control` - быстрый выбор активных устройств прямо сейчас.
- `Devices` - полный список всей подключенной техники по категориям.
- `Profiles` - создание, редактирование и активация профилей комбинаций устройств.

## Manual Control

- Позволяет быстро переключить активный микрофон.
- Позволяет быстро переключить активные наушники, колонки или другой аудиовыход.
- Позволяет переключить системный аудиовыход macOS.
- Показывает текущее активное устройство для каждой поддерживаемой системной роли.
- Ручное переключение не меняет сохраненные профили.
- После ручного переключения активное состояние переходит в режим `Manual Control`.
- В режиме `Manual Control` приложение все равно может удерживать выбранные вручную устройства как системные default.

## Devices

- Группирует устройства по категориям.
- Показывает краткие характеристики прямо в списке.
- Позволяет быстро понять, что именно сейчас подключено к Mac.
- Отдельно показывает устройства, которые удалось обнаружить, но не удалось уверенно классифицировать.
- Обновляет список при подключении и отключении устройств.

## Profiles

- Позволяет создать неограниченное количество профилей.
- Каждый профиль хранит выбранные устройства для нужных ролей.
- Профиль можно переименовать.
- Профиль можно редактировать.
- Профиль можно удалить.
- Профиль можно активировать из окна приложения.
- Профиль можно быстро активировать из menu bar.
- Когда окно приложения видно в Dock, профиль можно быстро активировать через Dock menu.
- Активный профиль постоянно поддерживается приложением в системе.

## Что хранит профиль

- Название профиля.
- Выбранный default input device.
- Выбранный default output device.
- Выбранный default system output device.
- Предпочтительная камера, если macOS или приложение-потребитель позволяет ее выбрать.
- Предпочтительные Bluetooth/HID/USB-устройства как часть ожидаемой конфигурации рабочего места.
- Флаг автоподдержания системных default-устройств.
- Дату создания и дату последнего изменения.

## Главная проблема, которую решает приложение

- macOS и приложения часто меняют default-устройства после подключения новой периферии.
- Например, после подключения AirPods системный микрофон может переключиться с USB-микрофона на микрофон AirPods.
- MacPeripheralHub следит за изменениями подключенных устройств и системных default-настроек.
- Если активный профиль требует конкретный микрофон или аудиовыход, приложение возвращает macOS к выбранным устройствам.
- Это позволяет держать стабильные настройки для звонков, записи, стриминга, рабочих сценариев и многопериферийных сетапов.

## Техническая идея

- Swift/AppKit отвечает за окно приложения, menu bar, Dock menu и интеграцию с macOS UI.
- C отвечает за основную логику приложения: модели устройств, профили, правила выбора, SQLite-хранилище, проверку состояния и тестируемый core.
- SQLite хранит профили, выбранные устройства, aliases и историю обнаруженных устройств.
- CoreAudio используется для обнаружения аудиоустройств и установки default input/output/system output.
- AVFoundation используется для камер.
- CoreGraphics/IOKit используются для дисплеев и системной периферии.
- IOKit HID используется для клавиатур, мышей, трекпадов и похожих устройств.
- IOBluetooth используется для Bluetooth-устройств, где это доступно.
- Приложение использует watchers/listeners и periodic reconciliation, чтобы возвращать нужные default-устройства после системных сбросов.

## Важные ограничения macOS

- macOS позволяет системно переключать default audio input, default audio output и default system output.
- macOS не дает универсального системного default-выбора камеры для всех приложений так же надежно, как для аудио.
- Для камер приложение хранит предпочтение и показывает его в профиле, но фактический выбор камеры зависит от конкретного приложения.
- Для клавиатур, мышей, USB-хабов и большинства HID/USB-устройств приложение может отслеживать наличие и характеристики, но не всегда может сделать устройство "активным" системно.
- Главный принудительно поддерживаемый сценарий версии 1.0.0 - стабильное удержание аудиовхода, аудиовыхода и системного аудиовыхода.
